# gatsby-mdx-embed

### All your favourite providers in one place ðŸ˜Ž!

`gatsby-mdx-embed` exposes a custom `MdxProvider` called `MdxEmbedProvider` which you can wrap around your content. You can then use the `gatsby-mdx-embed` components in your `.mdx` files... without import!

The following `gatsby-mdx-embed` components are currently available

#### CodeSandbox

| Props | Required | Type   |
| ----- | -------- | ------ |
| id    | yes      | string |

```js
<CodeSandbox id="react-new" />
```

<CodeSandbox id="react-new" />
<br />
