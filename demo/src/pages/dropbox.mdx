## Basic usage

<Playground>

...

You'll need to create an **app** to generate an **app key** in Dropbox, follow these [instructions](https://www.dropbox.com/developers/chooser). Once you have your **app key** you can add it to the plugin options of `gatsby-mdx-embed`

```js
// gatsby.config
module.exports = {
  ...
  plugins: [
    {
      resolve: `@pauliescanlon/gatsby-mdx-embed`,
      options: {
        dropboxAppKey: "YOUR_APP_KEY",
      },
    },
  ]
  ...
}
```
