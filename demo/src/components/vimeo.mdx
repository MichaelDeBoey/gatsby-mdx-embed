# Vimeo

Display a Vimeo video by including the component in your `.mdx`. An `id` is required. There are a number additional props available.

<Props of={Vimeo} />

## Basic usage

<Playground>
  <Vimeo id="38445453" />
</Playground>

## Skip to

<Playground>
  <Vimeo id="38445453" skipTo={{ h: 0, m: 1, s: 15 }} />
</Playground>
