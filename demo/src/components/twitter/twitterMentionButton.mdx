# TwitterMentionButton

Display a TwitterMentionButton by including the component in your `.mdx`. The `username` prop is required.

<Props of={TwitterMentionButton} />

## Basic usage

<Playground>
  <TwitterMentionButton username="PaulieScanlon" />
</Playground>

## Size

<Playground>
  <TwitterMentionButton username="PaulieScanlon" size="large" />
</Playground>
