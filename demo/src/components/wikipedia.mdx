# Wikipedia

Display a Wikipedia page by including the component in your `.mdx`. The `wikipediaLink` prop is required.

<Props name="Wikipedia" of={Wikipedia} />

## Basic usage

<Playground>
  <Wikipedia wikipediaLink="Tim_Berners-Lee" />
</Playground>

## Height

<Playground>
  <Wikipedia wikipediaLink="Tim_Berners-Lee" height="400px" />
</Playground>
