# TwitterFollowButton

Display a TwitterFollowButton by including the component in your `.mdx`. An `id` is required. There are a number additional props available.

<Props of={TwitterFollowButton} />

## Basic usage

<Playground>
  <TwitterFollowButton username="PaulieScanlon" />
</Playground>

## Show followers

<Playground>
  <TwitterFollowButton username="gatsbyjs" showFollowers={true} />
</Playground>
